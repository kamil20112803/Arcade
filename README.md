В проекте реализовано три уровня
в первом уровне вам надо победить 20 монстров, чтобы перейти на второй уровень
во 2-м уровне вам надо победить еще 20 монстров, чтобы перейти на финальный уровень
на 3-м уровне вы можете побеждать бесконечное количество монстров, и заканчивать игру по своему желанию.
в игре есть система учета рекордов
если мостры победят вас, вы проиграете
